## :information_source: How To Use

---

```bash
# Create a project using this boilerplate
$ yarn create next-app -e https://github.com/lpaulovt/nextjs-boilerplate

# ✔ What is your project named?
$ project-name

# Install dependencies
$ yarn install

# Run the app
$ yarn dev

# Run tests
$ yarn test:watch

# Run storybook
$ yarn storybook

# Run build
$ yarn build
```
